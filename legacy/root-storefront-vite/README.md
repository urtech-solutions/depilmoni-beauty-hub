# Storefront legado da raiz

Este diretorio guarda a antiga vitrine `Vite + React Router` que existia na pasta raiz.

Ela foi arquivada porque o projeto foi consolidado no monorepo oficial:

- `apps/web`: unica vitrine publica oficial
- `apps/cms`: admin do Payload CMS
- `apps/commerce`: base de comercio preparada para Medusa

Motivo do arquivamento:

- evitar duas vitrines publicas concorrentes
- preservar a referencia visual aprovada pela equipe
- manter a raiz limpa, atuando apenas como orquestradora do workspace
